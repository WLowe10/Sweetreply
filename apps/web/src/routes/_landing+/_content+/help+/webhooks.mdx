# lead.created

This event is sent whenever a new lead is detected in Sweetreply

```json
{
	"event": "lead.created",
	"data": {
		// the id of the lead
		"id": "lead_123",

		// the username of the lead poster
		"name": "John Doe",
		"email": ""
	}
}
```
